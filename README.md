<h2 align="center">how to use?</h2>

   ```
   - Clone the repository:
   $ git clone https://github.com/felipecarvalhogodoi98/Happy Happy

   - Enter the directory:
   $ cd Happy
   
   - Install dependecies:
   $ npm install

   - Execute the aplication:
   $ npm start

   - Open in your browser:
   http://localhost:5500/
   ```
